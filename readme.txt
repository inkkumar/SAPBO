DESCRIPTION

This Java Server Pages (JSP) sample demonstrates how to
change a Business Objects Enterprise User Name.
________________________________________________________________

FILES

index.html         - collects information necessary to run the sample
ChangeUsername.jsp - calls IUser.setTitle() to change the BOE user name
more_info.html     - links to our website as well as disclaimer
style.css          - formatting info

________________________________________________________________ 

INSTALLATION

1. Setup an application context on your application server to host a web application.

2. Copy the contents of the Java_Enterprise_BE115_Change_Username sample folder into your web application's context root.

3. Launch the index.html file in a browser window
